# KIRA — Mini ChatGPT de Pesquisa Científica, Tecnologia e Mercado Financeiro

KIRA é uma assistente em Python focada em pesquisa profunda, resumo objetivo, tradução automática, cálculos simbólicos, tecnologia, física e mercado financeiro.

## Instalação

```bash
cd /home/dev-0/Documentos/KIRA
python3 -m venv venv
source venv/bin/activate
pip install --upgrade pip setuptools wheel
pip install -r requirements.txt
```

## Rodar

```bash
python main.py
```

## Comandos

```text
/pesquisar sua pergunta
/cientifico tema científico ou físico
/tecnologia tema de tecnologia
/financeiro PETR4.SA ou BTC-USD ou AAPL
/calcular integral(x**2, x)
/traduzir en texto em português
/ajuda
/sair
```

## Exemplos

```text
/pesquisar o impacto da inteligência artificial na educação
/cientifico fusão nuclear tokamak
/tecnologia melhores bancos vetoriais para IA
/financeiro PETR4.SA
/calcular diff(sin(x)*exp(x), x)
/traduzir en computação quântica aplicada à segurança
```
